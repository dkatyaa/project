# Внедрение новой механики оплаты и аналитика клиентских сегментов

**Цель:** \
Проанализировать влияние новой механики оплаты на ключевые продуктовые метрики, сегментировать клиентов для персонализированного маркетинга и автоматизировать расчёт аналитических показателей.

**Задачи:**
1. Провести A/B тест. \
В ходе тестирования одной гипотезы целевой группе была предложена новая механика оплаты услуг на сайте, у контрольной группы оставалась базовая механика.
*Стек: Python (pandas, numpy, seaborn, scipy.stats, matplotlib.pyplot)*
2. Сегментировать клиентов для персонализированного маркетинга. \
CRM-отдел строит систему сегментации клиентов для триггерных рассылок на основе истории покупок. Сегмент клиента определяет тип маркетинговой коммуникации. \
*Стек: SQL (JOIN, CTE, оконные функции, агрегация, группировка)*
3. Написать функции для автоматической подгрузки информации из файла и пересчета метрик и функции, которая строит графики по получаемым метрикам (для задачи 1) \
*Стек: Python*

### Что было сделано:
- Проведен предварительный анализ данных (EDA);
- Выбраны и посчитаны метрики (CR, ARPU, ARPPU) в контрольной и тестовой группе;
- Проверена статистическая значимость изменений в метриках у групп с помощью статистических тестов (z-test, t-test, бутстрап, тест Левена);
- Построены графики для сравнения показателей двух групп;
- Сформулированы выводы и рекомендации;
- Написан SQL-запрос (с использованием CTE, JOIN, оконных функций, агрегации, группировки);
- Выполнена сегментация клиентов по покупательскому поведению;
- Написана функция для автоматической выгрузки данных, пересчета метрик и визуализации результата.
