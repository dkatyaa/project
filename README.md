# Внедрение новой механики оплаты и аналитика клиентских сегментов

**Цель:** \
Проанализировать влияние новой механики оплаты на ключевые продуктовые метрики, сегментировать клиентов для персонализированного маркетинга и автоматизировать расчёт аналитических показателей.

**Проект состоит из 3 частей:**
1. **A/B тест**. В ходе тестирования одной гипотезы целевой группе была предложена новая механика оплаты услуг на сайте, у контрольной группы оставалась базовая механика. Необходимо проанализировать итоги эксперимента и сделать вывод, стоит ли запускать новую механику оплаты на всех пользователей \
*Стек: Python (pandas, numpy, seaborn, scipy.stats, matplotlib.pyplot)*
2. **Cегментация клиентов для персонализированного маркетинга (SQL)**. CRM-отдел строит систему сегментации клиентов для триггерных рассылок на основе истории покупок. Сегмент клиента определяет тип маркетинговой коммуникации. \
*Стек: SQL*
4. Написание **функции** для автоматической подгрузки информации из файла и пересчета метрик и функции, которая строит графики по получаемым метрикам (для части 1) \
*Стек: Python*
